Roadmap

This roadmap is the conclusion of it was hosting Stack Programming Community 1st Conference when March 16, 2024.

Language Objectives

We decided Stack aims for convenient shell script language.
Plan is to enhance and increase many convenient commands as shell script.
For example, file processing, get computer system data, and web scraping, etc.

And, we decided to implement the Stack object-oriented system like a Common-Lisp’s one. That’s Stack become can handle user-defined data structure.

Community Organization

We decided to presentation the Stack at many events of technical and academical to spread to the world.
