Lenguaje de programación Stack

Lenguaje de programación Stack

El potente lenguaje de secuencias diseñado con un enfoque orientado a pilas para una ejecución eficiente.

Repositorio en GitHub ¡Noticias!Ahora se puede controlar el portapapeles

Características

  • Modelo de ejecución eficiente orientado a pilas: ejecución muy rápida como lenguaje de secuencias, también fácil de depurar
  • Sintaxis clara, fácil de entender y elegante: separa datos y comandos con espacios, sin ambigüedad
  • Conjunto rico de comandos convenientes y altamente extensibles: desde cálculos simples hasta expresiones regulares y solicitudes HTTP
  • Sistema de tipos de datos dinámico y flexible: no es necesario preocuparse por el sistema de tipos confuso normal

Código de ejemplo

Este es un código de ejemplo para resolver el problema FizzBuzz en el lenguaje de programación Stack. ¿No es elegante? 

1 101 1 range (i) (
    (FizzBuzz) (
        (Fizz) (
            (Buzz) i
            i 5 mod 0 equal if
        )
        i 3 mod 0 equal if
    )
    i 15 mod 0 equal if
) map ( ) join print

Puede utilizar el entorno de ejecución en línea para probar este script del lenguaje de programación Stack.
Si desea ejecutar el script del lenguaje de programación Stack en su computadora, debe instalar el intérprete.

Instalación

Puede instalar el intérprete del lenguaje de programación Stack mediante versiones regulares en su computadora.
Si tiene los comandos "git" y "cargo", también puede ejecutar los siguientes comandos para instalar el lenguaje de programación Stack.
Así puede obtener las últimas funciones antes de las versiones regulares. Sin embargo, las funciones no son estables. 

git clone https://github.com/stack-community/stack-lang.git
cd stack-lang
cargo build --release
cargo install --path .
