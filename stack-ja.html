Stack プログラミング言語

効率的な実行を目指したスタック指向のアプローチで設計された強力なスクリプト言語です。

GitHub リポジトリ ニュース！「cls」コマンドを実装し、コンソールの表示をクリアできるようになりました。

特徴

  • 効率的なスタック指向の実行モデル：非常に高速なスクリプト言語の実行、デバッグも容易
  • 明確で理解しやすくエレガントな構文：データとコマンドをスペースで区切り、曖昧さがありません
  • 便利で高度に拡張可能なコマンドが豊富：単純な計算から正規表現や HTTP リクエストまで
  • 動的で柔軟なデータ型システム：通常の混乱する型システムを気にする必要はありません

サンプルコード

これは Stack プログラミング言語で FizzBuzz 問題を解決するサンプルコードです。エレガントではありませんか？ 

1 101 1 range (i) (
    (FizzBuzz) (
        (Fizz) (
            (Buzz) i
            i 5 mod 0 equal if
        )
        i 3 mod 0 equal if
    )
    i 15 mod 0 equal if
) map ( ) join print

Stack プログラミング言語のスクリプトを試すにはオンライン実行環境を使用できます。
コンピューターで Stack プログラミング言語のスクリプトを実行したい場合は、インタプリタをインストールする必要があります。

インストール

Stack プログラミング言語のインタプリタは、定期的にリリースされています。
"git" と "cargo" コマンドを使用できる場合は、以下のコマンドを実行して Stack プログラミング言語をインストールすることもできます。
これにより、定期的なリリース前に最新の機能を取得できますが、その機能は安定していません。 

git clone https://github.com/Stack-Programing-Community/Stack-Programing-Language.git
cd Stack-Programing-Language
cargo build --release
cargo install --path .
