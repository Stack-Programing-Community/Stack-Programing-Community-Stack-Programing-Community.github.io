Stack プログラミング言語

効率的な実行を目指しスタック指向のアプローチで設計された強力なスクリプト言語

GitHub リポジトリ ニュース！ファイルパスの表現がクロスプラットフォーム互換性を持つようになりました

特徴

  • 効率的なスタック指向の実行モデル：スクリプト言語としては非常に高速に実行でき、デバッグも簡単
  • 明確で理解しやすくエレガントな構文：データとコマンドをスペースで区切り、曖昧さがありません
  • 便利で高度に拡張可能なコマンドが豊富：単純な計算から正規表現や HTTP リクエストまで
  • 動的で柔軟なデータ型システム：通常の混乱する型システムを気にする必要はありません

学ぶ

Stackプログラミング言語を学ぶにはチュートリアルが初心者でも分かりやすいのでお勧めです。

サンプルコード

これはStackプログラミング言語で FizzBuzz 問題を解くサンプルコードです。エレガントではありませんか？ 

1 101 1 range (i) (
    (FizzBuzz) (
        (Fizz) (
            (Buzz) i
            i 5 mod 0 equal if
        )
        i 3 mod 0 equal if
    )
    i 15 mod 0 equal if
) map ( ) join print

Stackプログラミング言語のスクリプトを試すにはオンライン実行環境を使用できます。
パソコン上でStackプログラミング言語のスクリプトを実行したい場合は、インタプリタのインストールが必要です。

インストール

Stackプログラミング言語のインタプリタは、定期的にリリースされています。
"git" と "cargo" コマンドを持っている場合は、以下のコマンドを実行してStackプログラミング言語をインストールすることもできます。 これにより、定期的なリリース前に最新の機能を取得できますが、その機能は安定していません。 

git clone https://github.com/stack-community/stack-lang.git
cd stack-lang
cargo build --release
cargo install --path .
