Why Stack ?

Stack is used numerous development project. that reasons are bolow 4 point.

  • Efficient execution: stack-base model is recruitmented as execution logic, it can run fast
  • Dynamic types system: powerful meta-programming features helps you to write the code
  • Multi-paradigm language: support many programming paradigm from functionnal to object-oriented
  • Variety distribution: for many use-cases from use shell script to build web server, etc
