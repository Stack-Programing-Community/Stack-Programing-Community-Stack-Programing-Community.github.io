Язык программирования Stack

Мощный сценарный язык, разработанный с учетом стекового подхода для эффективного выполнения.

Репозиторий GitHub Новости!Теперь можно контролировать буфер обмена

Особенности

  • Эффективная стековая модель выполнения: очень быстрое выполнение как сценарного языка, также легко отлаживать
  • Четкий, понятный и элегантный синтаксис: разделение данных и команд пробелами, отсутствие неоднозначности
  • Богатый набор удобных и высокорасширяемых команд: от простого вычисления до регулярных выражений и HTTP-запросов
  • Динамичная и гибкая система типов данных: вам не нужно беспокоиться о типовой системе, которая часто вызывает путаницу

Учить

Пример кода

Это пример кода для решения проблемы FizzBuzz на языке программирования Stack. Элегантно, не так ли? 

1 101 1 range (i) (
    (FizzBuzz) (
        (Fizz) (
            (Buzz) i
            i 5 mod 0 equal if
        )
        i 3 mod 0 equal if
    )
    i 15 mod 0 equal if
) map ( ) join print

Вы можете использовать онлайн-среду выполнения, чтобы попробовать запустить этот сценарий на языке Stack.
Если вы хотите запустить сценарий на языке Stack на своем компьютере, вам нужно установить интерпретатор.

Установка

Вы можете установить интерпретатор языка Stack, выполнив регулярные выпуски на вашем компьютере.
Если у вас есть команды "git" и "cargo", вы также можете выполнить следующие команды для установки языка программирования Stack.
Таким образом, вы можете получить последние функции перед регулярными выпусками. Однако эти функции нестабильны. 

git clone https://github.com/stack-community/stack-lang.git
cd stack-lang
cargo build --release
cargo install --path .

Язык программирования Stack - это свободное и открытое программное обеспечение под лицензией MIT.
Любой может использовать, копировать, изменять, объединять, публиковать, распространять, лицензировать, продавать язык программирования Stack.
