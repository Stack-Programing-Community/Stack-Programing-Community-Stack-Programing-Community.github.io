Stack 编程语言

设计为栈导向方法以实现高效执行的强大脚本语言。

GitHub 仓库 新闻！ 实现了 "cls" 命令，可以清除控制台的显示。

特性

  • 高效的栈导向执行模型：作为脚本语言，执行速度非常快，也很容易调试
  • 清晰、易于理解且优雅的语法：用空格分隔数据和命令，没有歧义
  • 丰富的便利且高度可扩展的命令：从简单计算到正则表达式和 HTTP 请求
  • 动态且灵活的数据类型系统：您不必担心常规混乱的类型系统

示例代码

这是用 Stack 编程语言解决FizzBuzz问题的示例代码。是不是很优雅？ 

1 101 1 range (i) (
    (FizzBuzz) (
        (Fizz) (
            (Buzz) i
            i 5 mod 0 equal if
        )
        i 3 mod 0 equal if
    )
    i 15 mod 0 equal if
) map ( ) join print

您可以使用在线执行环境尝试运行 Stack 编程语言的这个脚本。
如果要在计算机上运行 Stack 编程语言的脚本，则必须安装解释器。

安装

您可以通过定期发布 在您的计算机上安装 Stack 编程语言的解释器。
如果您有 "git" 和 "cargo" 命令，也可以运行以下命令安装 Stack 编程语言。
这样您可以在定期发布之前获取最新功能，但这些功能不稳定。 

git clone https://github.com/stack-community/Stack-Programing-Language.git
cd Stack-Programing-Language
cargo build --release
cargo install --path .
