Stack Programming Language

The powerful script language designed with a stack oriented approach for efficient execution.

Github repository News!　implemented command "case", it became can change string style case

Features

  • Efficient stack oriented execution model： very fast execution as script language, easy to debug too
  • Clear, easy to understand and elegant syntax： separate data and commands with spaces, no ambiguity
  • Rich of convenient and highly extensible commands： from simple calculate to Regex and HTTP request
  • Dynamic and flexible data types system： you don't have to worry for normal confusing type system

Sample code

This is a sample code to solve FizzBuzz problem in Stack programming language. Isn't it elegant? 

1 101 1 range (i) (
     (FizzBuzz) (
         (Fizz) (
             (Buzz) i
             i 5 mod 0 equal if
         )
         i 3 mod 0 equal if
     )
     i 15 mod 0 equal if
) map () join print

Install

You can install Stack programming language in the your computer at Regularly Release.
If you have commands "git" and "cargo", you can run below commands to install stack programming language.
that's you can get latest functions before the regularly releases. but, the functions are not stable. 

git clone https://github.com/Stack-Programing-Community/Stack-Programing-Language.git
cd Stack-Programing-Language
cargo build --release
cargo install --path .
